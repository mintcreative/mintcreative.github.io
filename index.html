openMampHtdocsiTerm

A simple AppleScript to open iTerm in the /Applications/MAMP/htdocs folder

MAC OS
Instructions
Open the openMampHtdocsiTerm.sct file in the AppleScript editor - a double click should do it.
Inside the editor select File->Export. In the dialogue box, give this App a name (maybe 'open htdocs').
From the first dropdown menu (File Format) select Application.
Select a location to save the App and click Save.

Enjoy your new shortcut to open iTerm at your local htdocs folder.
You're welcome.
